# Customer Behavior Analytics Pipeline

This project demonstrates a complete data engineering pipeline
for analyzing customer purchasing behavior.

## Tech Stack

Python
PySpark
PostgreSQL
Apache Airflow
Power BI

## Pipeline Architecture

1. Extract transaction data
2. Clean data using PySpark
3. Transform customer metrics
4. Load metrics into PostgreSQL
5. Automate pipeline with Airflow
6. Visualize insights in Power BI

## Key Insights

Top customers
Customer lifetime value
Purchase trends
Regional sales performance
